# rti-project
RTI community robotics project simulating a smart factory warehouse using ROS2 &amp; Gazebo
Built by a community of robotics learners from across India

## Features
- Multiple AGVs
- Dynamic job queue
- Collision avoidance
- Emergency rerouting

## Tech Stack
- ROS2 (Foxy/Humble)
- Gazebo
- Python / C++
- GitHub for collaboration
